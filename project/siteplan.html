Site Plan: Local Travel Guide for Rexburg, Idaho

Site Name

Proposed Site Name: Discover Rexburg

The name "Discover Rexburg" was chosen because it reflects the purpose of the site: helping visitors and locals uncover the best attractions, restaurants, and experiences in Rexburg.

Site Purpose

The purpose of this website is to provide a comprehensive travel guide for Rexburg, Idaho. It will offer information on popular attractions, hidden gems, and local restaurants, as well as a dynamic filtering feature to help users easily find what theyâ€™re interested in. The site will also feature a "Save for Later" list and a feedback form to enhance user interaction.

Scenarios

  • Scenario 1: A college student wants to find affordable restaurants near campus.
  • Scenario 2: A family visiting Rexburg is looking for kid-friendly attractions.

Color Schema

Primary Color: #1e73be (used for headings and navigation links)

Secondary Color: #5d5c61 (used for background of the header and footer)

Accent Color: #ff9800 (used for buttons and highlights)

Typography

Primary Font: Roboto (used for body text)

Secondary Font: Lora (used for headings)
