Site Plan: Rexburg Travel Guide

Site Name

Rexburg Travel Guide: This name reflects the focus on providing information about attractions, restaurants, and activities in Rexburg, Idaho.

Site Purpose

The purpose of the website is to serve as a local travel guide for Rexburg, Idaho. It helps visitors and residents discover things to do, places to eat, and attractions to visit. The site will include features such as dynamic filtering, a "Save for Later" list, and a feedback form.

Scenarios

  • What are the best restaurants in Rexburg?
  • What are some must-see attractions in Rexburg?

Color Schema

The selected color schema includes the following:

  • Primary Color: #4CAF50 – Used for headings, buttons, and accents.
  • Secondary Color: #f4f4f4 – Used as the background for sections.
  • Text Color: #333333 – Used for body text.

Typography

The chosen typography includes:

  • Heading Font: Poppins, sans-serif
  • Body Font: Roboto, sans-serif

Wireframes

Below are the wireframe sketches for the home page layout:

  • Mobile View: A simple vertical layout with a navigation menu at the top, followed by featured sections and a footer.
  • Desktop View: A grid-based layout with a navigation bar, featured content, and a multi-column section for categories.

Note: The wireframes are attached as sketches in the project folder.
